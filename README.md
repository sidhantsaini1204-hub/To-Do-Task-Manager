## Modern To-Do Task Manager

A clean and responsive task manager built using HTML, CSS, and JavaScript. It helps users create, edit, complete, and delete daily tasks with a modern user interface.

Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Search tasks
- Filter completed and pending tasks
- Custom theme colors
- Responsive design

Technologies Used
- HTML5
- CSS3
- JavaScript

Project Structure
- index.html
- style.css
- script.js
- README.md

How to Use
- Download or clone the repository.
- Open `index.html` in any modern browser.
- Start managing your tasks.

Future Enhancements
- User login
- Cloud storage
- Notifications
- Drag and drop tasks
